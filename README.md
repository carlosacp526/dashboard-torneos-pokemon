# Dashboard de Torneos Pokémon

Dashboard interactivo para gestión y análisis de torneos Pokémon.

## Características

- Estadísticas de jugadores
- Clasificación por eventos y tiers
- Historial de combates
- Ranking mundial
- Perfiles de jugadores

## Instalación local
```bash
pip install -r requirements.txt
streamlit run app.py
```
